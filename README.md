# Student Performance Analysis & preprocessing

A comprehensive Python and Jupyter Notebook repository dedicated to student academic data processing . This project implements data cleaning pipelines alongside multi-layered Artificial Neural Network (ANN) architectures built from scratch.

## 📂 Repository Structure & Project Files
- **`students_data.csv`**: The primary student dataset containing features like names, age, and academic marks.
- **`file1.py`**: Automated data preprocessing pipeline handling dynamic missing values, custom feature grading engineering, and data normalization.
- **`AnnforOrwithout... / AnnForXorwithhi...`**: Customized Python scripts implementing Artificial Neural Network (ANN) logic models to solve classic boolean logic classifications (OR/XOR gates).
- **`Minist.ann.py`**: Neural network script tailored for fundamental dataset recognition tasks.
- **`notebook.ipynb`**: Interactive Jupyter Notebook containing descriptive dataset analysis and live visualization checkpoints.
- **`normalized_marks.csv`**: Cleansed and exported database ready for machine learning feature training.


## 🛠️ Tech Stack
- Python 3
- Jupyter Notebook
- Pandas
- NumPy
